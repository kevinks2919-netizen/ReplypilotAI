## Suggested YouTube Description

```text
This video demonstrates how ReplyPilot AI uses Google OAuth Gmail scopes.

ReplyPilot AI uses gmail.readonly to show recent inbound Gmail message previews inside the connected accounts dashboard. It uses gmail.send only when the authenticated user manually reviews a generated reply and clicks Send. The app does not delete Gmail messages, does not send replies automatically to all senders, and does not use Gmail data for advertising or resale.
```
